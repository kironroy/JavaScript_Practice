☑️ Closures are created in JavaScript automatically

Don't need to return a function from another function to create a closure.

A closure gives a function access to all variables of its parent function, even after the parent function has returned.

The function keeps a reference to its outer scope which preserves the scope chain throughout the lifetime of the closure.

👉 Or you can think of it like a 🎒 backpack. A function carries this 🎒 backpack wherever it goes. This backpack 🎒 has all the variables that were present in the environment where the function was created.
