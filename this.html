this keyword: is a special variable that is created for every execution context (every function). It takes a value of (points to) the "owner" of the function. in which the keyword is used.

Execution Context

  • Variable environment
  • Scope chain
  • this

4 different ways a function can be called

  • Method: A function attached to an object. this points to the object that is calling the method
  • Simple: this is undefined in strict mode
  • Arrow: this will be parent function
  • Event listener:this = to the DOM element that the handler is attached too.

this does not point to the function itself nor the variable environment.
